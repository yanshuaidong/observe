
### 二、策略想法字段（模板定义，说明每条记录里需要填哪些字段）

```
策略想法字段:{

  品种与操作:{
    品种名字：（国泰君安推荐的品种，比如：黄金期货、碳酸锂期货、螺纹钢期货...）
    合约代码：（可选，比如：AU2506、LC2506，方便以后精确追踪）
    品种方向：（做多 / 做空 / 观望）
    有没有操作：（可选值：未操作 / 开仓 / 平仓 / 加仓 / 减仓 / 止损 / 止盈）
    操作价格：（可选，实际成交价格）
    操作数量：（可选，手数）
    止损价：（可选，设置的止损位）
    止盈价：（可选，设置的止盈位）
    操作的原因：（一段文案，比如"大单连续3天净买入增多，跟随做多"）
    有没有后悔：是/否；
    后悔的原因：（可选，一段文案）
  }

  推荐来源:{
    国泰君安推荐日期：（什么时候推荐的，比如"20260330"）
    国泰君安推荐理由：（可选，一段文案，国泰君安给的推荐原因摘要）
    国泰君安推荐方向：（做多 / 做空 / 中性）
  }

  大单数据:[
    {
      日期："20260328"；
      大单净买入量：1200；（正数=净买入，负数=净卖出）
      较前一日变化："增多"；（可选值：增多 / 减少 / 持平）
    }，
    {
      日期："20260329"；
      大单净买入量：1800；
      较前一日变化："增多"；
    }，
    {
      日期："20260330"；
      大单净买入量：2500；
      较前一日变化："增多"；
    }
    ...至少记录3天，用来判断是否满足连续3天同向变化
  ]

  大单信号判断:{
    是否连续3天同向变化：是/否；（核心开仓条件）
    变化方向："连续增多"；（可选值：连续增多 / 连续减少 / 不满足）
    信号强度：（可选，1-5，1最弱5最强，根据增减幅度主观判断）
    信号备注：（可选，比如"第3天增幅明显放大，信号较强"）
  }

  消息面:[
    {消息："国泰君安发布看多黄金研报"，来源："国泰君安"}，
    {消息："美联储暗示降息"，来源："新华社"}，
    ...可以有多条
  ]

  情绪面:{
    我的情绪：（可选值：贪婪 / 恐惧 / 犹豫 / 冷静 / 兴奋 / 焦虑 / 无感）
    情绪强度：（可选，1-5，1最弱5最强）
    跟随信心：（可选，1-5，1最没信心5最有信心，对这次跟随信号的信心）
    情绪备注：（可选，比如"信号很明确，跟着走就行"、"虽然信号满足但总觉得不太对"）
  }

}
```
